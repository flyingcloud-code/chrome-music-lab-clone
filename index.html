Chrome Music Lab local index

This page serves the downloaded googlecreativelab/chrome-music-lab repository from one local server. The original repo is split into independent experiments, and the legacy 2016 build chains have been upgraded to run on Node 22 with Python 3.

Build note: the webpack experiments now use webpack 5 and Dart Sass instead of node-sass. Spectrogram now uses a small modern Node build script instead of gulp 3/Compass. Sass still reports deprecation warnings for old @import and slash division syntax, but the apps build successfully.
